Supervoxel Labeling


Only one algorithm needs to be fully labeled here.

Save your data to come back to later or Finish to move on to the overlaying step.


Click on a label to set as current.

Click or drag mouse across either canvas to label supervoxels.

  1. Choose labels and their colors, (Saving will save these settings).
  2. Label video. Save whenever needed.
  3. Hit finish and then move to the overlaying step.

Labels

  • Labels can be modified: add,remove,rename.
  • You can also save and reset them.
  • Use your best judgement on the level of detail to annotate
    and create a label for each segment.
  • While a label is selected, click the colorpicker to change it's color.

Rules

  • Each unique item of a class should have its own label,
    (i.e. person1, person2, etc...).
  • Make sure the entire video is labeled before finishing.

Tips

  • Depending on the hierarchy chosen, different amounts of the video will be colored.
    Start with the highest level and work down to the details.
    Skipping hierarchies is fine.
  • To change the frame click Prev, Next, Play, move the slider below the video, or scroll while the mouse is over the video.
  • *Important* Reset buttons do not save the action.
    Refresh the browser to restore the last save point.

Supervoxel Overlaying


  1. Choose an algorithm and a label source.
  2. Fix labels as needed.
  3. Save the output.
    4. Optional:
  4. Return and choose another algorithm.
    Use 'Output' as your labeled-frame source.
  5. Repeat as needed or even return from Post-Processing step.

General

  • Everything on the interface is essentially the same.
  • Use the DELETE label to remove labels that are less accurate than the original.
  • Use the right-most canvas to see what changes you have made.
  • Labels can be added but not removed.

Tips

  • Post-Processing is a frame-by-frame process. This is very tedious! Do your best using uspervoxels to label!

Post-Processing Labels


  1. Draw the correct labels on, frame-by-frame, using a resizable brush.
  2. Save changes before moving to the next frame.

Tips

  • Saving only effects the current frame. Watch the video. Make sure you have a general idea of what is going on and where the major innaccuracies are. Fix those, or start at the beginning and go through thoroughly frame-by-frame.
  • By switching frames unsaved changes are lost. The first time you try to, a warning will appear giving you the option to save or ignore. Neither option will perform the frame-switching action.
